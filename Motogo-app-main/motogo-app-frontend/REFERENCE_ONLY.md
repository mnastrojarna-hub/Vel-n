## PRAVIDLA:
- NEŠAHEJ na žádný soubor v této složce

## Struktura:
- css/ — design systém (main.css, elements.css, screens.css)
- js/ — API volání, UI logika, i18n překlady
- templates*.js — šablony všech 37 obrazovek
- data/ — motorky, assety, legální texty
- src/services/ — Supabase client, auth
